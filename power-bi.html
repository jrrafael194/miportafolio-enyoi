Dashboard Interactivo realizado con Power Bi

En este proyecto se realizo un dashboard interactivo con Power Bi, se utilizo la data
del proyecto de Google Sheet, pasando todos estos datos a una base de datos de entidad relacional
en este caso utilizamos MySql, y conectamos estos datos de MySql con Power Bi, se realizaron
dashboard interactivos de donantes, proveedores y utilidades.
