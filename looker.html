En este proyecto se realizo un dashboard interactivo con Looker Studio, se utilizo la data
del proyecto de Google Sheet, Aparte la fundacion nos proporciono nuevos datos que se le realizo ETL
y se agregaron a nuestro archivo ya existente. En este proyecto conectamos los datos de Google Sheet con Looker
Studio. Y se realizaron combinacion de datos de diferentes tablas para realizar calculos de la Utilidad.
Ademas se crearon dashboard interactivos de Donantes que representan los ingresos y de proveedores que
representan los gastos. Cada dashboard tiene diferentes tipos de graficos y KPIs para su analisis y estudio.
